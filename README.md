## Apprentice-Practice　　

### Overview

アプレンティスのクエストを管理するリポジトリです。

### Discription

#### ディレクトリ構成　　
- week○-○：期間毎の全てのクエストを管理するディレクトリです。
- quest-○：言語毎にクエストを管理するディレクトリです。
- quest_○：各クエストを管理するディレクトリです。

<pre>
root
├── week1-2
│    ├── quest-typescript
│    │   └── quest_1
│    │   └── quest_2 
│    └── quest-ruby
│        └── quest_3
└── week3-4
     ├── quest-typescript
     │   └── quest_1
     │   └── quest_3
     └── quest-ruby
         └── quest_2
</pre>

#### ブランチの運用規則  
- 命名規則：yymmdd-quest 230416_blackjack
- 日毎にブランチを切る。

#### コミットの規則
- 命名規則：bjg_xxxxx quest_blackjack
